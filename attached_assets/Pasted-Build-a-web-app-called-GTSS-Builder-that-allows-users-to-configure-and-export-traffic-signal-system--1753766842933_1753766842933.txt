Build a web app called GTSS Builder that allows users to configure and export traffic signal system data in a GTFS-like format (GTSS = General Traffic Signal Specification).
üíª Frontend:
Use React (with Vite or Next.js) or Vue 3 for the frontend.


Display a tabbed or side navigation layout with the following sections:


Agency Info


Signal Locations


Phases


Detectors


Preview & Export


Each section should include a table-style editor to input rows of data and add/delete/update entries.


Use local state or Pinia/Vuex/Redux as needed to manage form inputs and relationships between signals, phases, and detectors.


üì¶ Backend:
Use a simple Express.js (or Python Flask/FastAPI) server to:

Send it back for user download.
